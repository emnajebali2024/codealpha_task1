##  Tech Stack

| Layer    | Technology |
|----------|-----------|
| Frontend | HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5.3 |
| Backend  | Node.js, Express.js |
| Database | MongoDB + Mongoose |
| Auth     | JWT (JSON Web Tokens) + bcryptjs |

---

##  Quick Start

### 1. Install & Seed Backend
```bash
cd backend
npm install
node seeder.js      # Seeds 12 products + admin account
npm run dev         # Start dev server on :5000
```

### 2. Run Frontend
Open `frontend/index.html` in any browser (use Live Server in VS Code for best experience).
